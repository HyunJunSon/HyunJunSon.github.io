정적 블로그 만들기

정적 블로그를 만들기는 매우 쉽습니다. 정적 블로그는 웹 사이트의 콘텐츠를 미리 생성한 HTML 파일로 제공하는 웹 개발 방식입니다. 이를 위해서 Hexo와 같은 정적 사이트 생성기를 사용할 수 있습니다.

1. Hexo 설치

먼저, Hexo를 설치해야 합니다. 터미널에서 다음 명령을 실행하여 Hexo를 설치합니다:

npm install -g hexo-cli
